# Gedankenstreich Deploy

Dieses Repository enthält **nur** das statische Deploy (Intro + Under-Construction) für Strato.

## Lokaler Build

```bash
npm install
npm run build
```

Danach den Inhalt von `dist/` auf den Webspace hochladen.

## Inhalte

- `/` Intro-Seite (Auto-Weiterleitung + Skip)
- `/under-construction/` Under-Construction-Seite
